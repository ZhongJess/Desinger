# 平面設計師個人作品集頁面

依據 AdobeXD 設計稿以純 HTML / CSS 手刻完成的響應式靜態網頁。

## 🔗 Demo

https://zhongjess.github.io/test920/

## 📄 頁面內容

- **Hero** — 姓名、職稱、社群連結（Behance / Facebook / Instagram / YouTube）
- **About** — 個人簡介、學歷、經歷
- **My Brand "XXXD"** — 個人服飾品牌介紹與形象圖
- **Other Works** — 2016–2020 年平面設計作品集（海報、LOGO 等）
- **Contact** — 聯絡信箱與電話

## 🛠️ 使用技術

- HTML5 語意化標籤
- CSS3（Flexbox / Grid）
- 響應式設計（RWD）
- 無使用任何 CSS 框架，全程手刻

## 📁 專案結構
```
test920/
├── index.html    # 主頁面
├── style.css     # 樣式表
└── image/        # 圖片與 SVG icon 素材
```
